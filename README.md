## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/java-practice.git
   ```

2. Navigate to the directory:
   ```bash
   cd java-practice
   ```

3. Compile and run the Java files using:
   ```bash
   javac FileName.java
   java FileName
   ```

4. Explore, learn, and enjoy coding! 💻
